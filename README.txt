# sgbd-dataguard
O projeto consiste em implementar um Sistema de Gerenciamento de Banco de Dados simplificado baseado no modelo relacional. Não é o objetivo desse projeto criar um SGBD para ser utilizado em produção por sistemas de informação, mas “apenas” explorar os conceitos abordados nas disciplinas programação com linguagem C.

Funcionalidades implementadas:
1 -  Criar tabela;
2 - Listar tabelas existentes;
3 - Inserir dados numa tabela;
4 - Exibir tabela criada;
5 - Busca valor na tabela (parcialmente);
6 - Remover linha em uma tabela;
7 - Remover arquivo existente;

Funcionalidades extra:
1- Implementamos o uso do Makefile para agilizar o processo de compilação do programa.
Para compilar, baixe todos os arquivos para uma mesma pasta e, no terminal rode o camando "make" para invocar o arquivo Makefile e gerar o arquivo "dataguard". Posteriormente, execute o "dataguard" como o comando "./dataguard"
2- Uso do GIT / GITHUB para o versionamento do código.

O que fariamos diferente?
Implemetaríamos uma ou mais struct para resolver alguns problemas que tivemos na busca da funcionalidade 5 além de inserir a opção onde o usuário escolhe o tipo de dado a ser inserido, facilitando também sua listagem  e exclusão.

Como compilar o projeto:
Para compilar, baixe todos os arquivos (main.c, funcoes.c, funcoes.h, makefile e readme.txt) do projeto para uma mesma pasta e, no terminal rode o camando "make" para invocar o arquivo Makefile e gerar o arquivo "dataguard". Posteriormente, execute o "dataguard" como o comando "./dataguard".
